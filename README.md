# simple-crud-app

## Project setup
```
npm install
```

## Install JSON server
```
npm install -g json-server
```
## Run JSON server
```
cd .\src\api\
json-server recipes.json
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
